---

# Study overview

Late spring frost events represent acute climatic disturbances that can significantly disrupt vegetation phenology and reduce productivity. However, their spatial impacts are often heterogeneous and mediated by environmental gradients such as elevation and aspect.

Using high-resolution **Sentinel-2 NDVI time series**, this study evaluates how a late spring frost event in 2019 influenced vegetation productivity across a mountainous landscape.

Specifically, the study examines:

* **NDVI decline associated with frost exposure**
* **Post-frost recovery trajectories across subsequent years**
* **The role of topography (elevation and aspect) in mediating frost impacts**
* **Differences between frost-affected and non-frost-affected areas**

By combining remote sensing data with statistical modelling, the study provides insight into how **extreme climatic events propagate through forest ecosystems and influence recovery processes**.

# Code overview

The `code/` directory contains scripts used to:

* Process **Sentinel-2 NDVI time series data**
* Identify frost-affected areas using:
  * **Temperature-based statistical thresholds**
  * **NDVI-based change detection**
* Calculate **NDVI change before and after frost events**
* Aggregate NDVI at multiple temporal scales (e.g., weekly summaries)
* Integrate NDVI data with **topographic variables** (elevation, aspect)
* Fit statistical models (linear models and mixed-effects models) to:
  * Test differences between frost and non-frost areas  
  * Evaluate interannual variation in NDVI  
  * Assess topographic controls on frost sensitivity
* Generate figures and summary tables used in the manuscript and supplementary information

Scripts are organised sequentially to reflect the analytical workflow.

# Data dictionary (key variables)

The following variables are central to the analyses:

| Variable            | Description                                                                 |
|--------------------|-----------------------------------------------------------------------------|
| **NDVI**           | Normalised Difference Vegetation Index derived from Sentinel-2 imagery      |
| **Mean NDVI**      | Aggregated NDVI (e.g., weekly mean per location)                            |
| **Ndvi change**    | Change in NDVI before vs after frost event                                  |
| **Frost class**    | Classification of points as frost-affected or non-frost                     |
| **Green-up**        | Timing of spring phenology (start of season)                                |
| **Elevation band** | Categorical elevation classes                                               |
| **Aspect**         | Topographic aspect (continuous or transformed variable)                     |
| **ID**             | Unique spatial identifier for each sampling point                           |
